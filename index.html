Violet Wilhelm's Guide to Using Great Fonts

An Instruction Challenge for JOUR108

How to select awesome fonts for your design!

Time required: 1 hour
Awesome rating: 100% awesome

Are you struggling with your latest graphic design project? Are you tired of using the same bland, boring fonts? Well, here's the perfect guide to help you make your next project stand out from the rest!

  • Things You'll Need

  • Device with access to Adobe
  • Paper
  • Writing Tool
  • fontspace.com
  • Water (it's important to stay hydrated)

  1. Ideate

    Take a minute and think about the project you’re designing. What do you want it to communicate to its audience? Write a list of words that you want your final design to communicate. Ex: modern, creative, classic, elegant. For example, if you’re designing a poster for a punk rock concert, you won’t necessarily want a font that is in an elegant cursive style.

  2. Explore

    Open up Fontspace and search some of those words. Make sure to check the 100% free tab in order to find commercially available fonts. Look at the fonts presented. Select a few you like and write down the names so you can search them later.

    IMPORTANT: Check the "commercial use" button on Fontspace.

  3. Test

    Ask yourself some questions about the font:

  4. Is it readable?
  5. Is it available in the required characters for the project? Some fonts are all caps or lack numbers and special characters.
  6. Does it adhere to the hierarchy of the design? Does it work for a title, which can be more decorative, but is it too elaborate for body text? Vice versa, is it too plain for a title?
  7. Is it available in multiple weights? (Italic, Bold, Semibold, etc.) Some fonts are only available as is.
  8. MOST IMPORTANT: Does it communicate the message to your audience?

  • Download

    Download the font, which will appear in a zip fie. Unzip it and click on the font file to add it to the computer’s available fonts.

    Select "Open File."

    Select "Install."

  • Apply

    Open up your design in Adobe. The font should appear in the font list on your device. Open a text box in the design, select it and begin typing.

    Scroll through the font list and find yours.

    Type in the text box.

  • Reflect

    Does the font work as you intended? Does it communicate the desired message? If so, great! If not, go back and choose another font you liked from the exploring stage and test it out.
